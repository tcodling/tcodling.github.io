CSS Positioning

The difference between relative, absolute and fixed positioning

8 February 2020

Position: relative

An element positioned relatively will be placed on the screen relative to where it would show up normally in the flow of the page. The position is defined by four properties: top, right, bottom, and left. The element will move in the opposite direction to the direction you define. In the diagram below, the purple square has been moved 200px to the right from its original position. This is because in css it was given a value of 'left: 200px'.

Parent Container

Code snippet

.originalPosition { border: 2px green dashed; width: 100px; height: 100px; position: absolute; } #relativePositioned { border: 2px purple solid; width: 100px; height: 100px; position: relative; left: 200px; }

Position: absolute

An element with a position value of absolute will be treated as though it does not exist by other elements on the page. If you try to set left, top, right or bottom values, you will find it starts moving in relationship to the edge of the screen. However, if the parent container has a position property of relative, the absolute element will be placed in relation to the relative container. In the diagram below, the absolute positioned box has a left value of 0px and it has been placed right up against the edge of its container because we set it to relative!

Parent Container

Code snippet

.originalPosition { border: 2px green dashed; width: 100px; height: 100px; position: absolute; } #absolutePositioned { border: 2px purple solid; width: 100px; height: 100px; position: absolute; left: 0px; }

Position: fixed

An element with a position value of fixed is similar to an absolute element however it is always placed in relationship with the edge of the screen! The circle has been placed 950px from the left side of the screen and 300px from the bottom. A fixed position element also moves as you scroll. Thats why the circle has been following you around the page!

Parent Container

Code snippet

.originalPosition { border: 2px green dashed; width: 100px; height: 100px; position: absolute; } #fixedPositioned { border: 2px purple solid; width: 100px; height: 100px; position: fixed; left: 950px; bottom: 150px; }
